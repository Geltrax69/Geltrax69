## <img src="https://media.giphy.com/media/QssGEmpkyEOhBCb7e1/giphy.gif" width="28"> &nbsp; Featured Builds

> The résumé shows two. Here's the real shelf.

### 🏆 ScoreCast — Real-Time Sports Scoreboard
**Golang · Gin · WebSocket · PostgreSQL · React · TypeScript · Docker**
A broadcast-style live scoreboard. A Golang WebSocket hub syncs scores, timers and match events
**instantly** across scorer phones, admin panels and display screens. QR / 4-digit pairing with JWT
device tokens lets officials run a match from their phone. Event-driven logic for sets, timers,
undo/redo and completion.
`real-time` · `event-driven` · `live-matches`

### 🏫 Simpedu — School Management Platform (Web + Mobile)
**Next.js · React · TypeScript · Node · Express · MongoDB · FCM**
Multi-tenant school system live across **3 schools / 100–200 students** — records, staff, fee
workflows. RESTful APIs with JWT + role-based access control, Firebase Cloud Messaging for real-time
homework/announcement/fee alerts. Fixed session & token-refresh bugs for stability.
`multi-tenant` · `rbac` · `production`

### 🍽️ RestaurantOS — Instagram-themed QR Ordering
**React · Multi-tenant · Admin + Owner + Customer panels**
Admins mint restaurant-owner accounts; owners publish a full menu (images, prices, veg/spicy, badges);
customers get a live Instagram-style ordering site at `/r/:slug`.
`saas` · `qr-ordering`

### 📦 Inventory Management System
**Spring Boot · React · Vite · PostgreSQL · JWT**
Dashboard-driven inventory app with JWT auth and PostgreSQL persistence for products & stock workflows.
`full-stack` · `jwt`

### 👵 Care — Elderly Caregiving Booking App
**Flutter**
Families book trusted caregivers for at-home elderly care. In active development.
`mobile` · `flutter`

### 💸 BetterTrack AI — Expense & Group Settle-Up
**Splitwise × Revolut × Apple Wallet, with a Gemini assistant built in.**
AI-native expense, budget and group settle-up app.
`ai` · `fintech`

### 📈 Stock Market Sentiment & Price Predictor
**Python · Stacked LSTM**
Predicts next-day price direction from 5 years of OHLCV technical features (AAPL, MSFT, GOOGL, AMZN, NVDA).
`ml` · `lstm`

### 💼 Candidate / Job Portal — Application & Admin Dashboard
**Node · Express · EJS · MongoDB**
Server-rendered system for managing job applications with an admin dashboard.
`backend`

### 🥻 Saree Store · ⚡ Zeus Inventory UI · 🏠 3D House Web · 🎨 Portfolio
Vanilla-JS dynamic catalog · React+Zustand analytics dashboard · 3D interactive site · personal portfolios.
